## What is Prompt Engineering?

Prompt工程是指设计与优化提示词（Prompts）以获得更好AI输出的实践。

## Key Concepts

### 1. Clear Instructions
- 明确告诉AI要做什么
- 使用具体而非模糊的描述

### 2. Context
- 提供必要的背景信息
- 设定角色或场景

### 3. Examples
- 使用few-shot learning
- 提供输入-输出示例

### 4. Structure
- 使用格式化的输出
- 使用分隔符

### 5. Iterative Refinement
- 逐步优化提示词
- 测试不同版本
