# GitNexus Assistant

**代码库 → 知识图谱 → 理解代码**

## 功能

| 功能 | 说明 |
|------|------|
| 架构理解 | 代码是怎么组织的？ |
| 调用链 | 函数在哪里被调用？ |
| 影响分析 | 修改会影响哪些地方？ |
| 调试 | 为什么出错了？ |

---

## 使用场景

| 用户说 | 动作 |
|--------|------|
| "这是什么代码" | 架构分析 |
| "函数X在哪用" | 调用链查找 |
| "改这个会影响到" | 影响分析 |
| "为什么报错" | 调试分析 |

---

## 设计原则

- **s05**: 按需加载代码知识
- **s04**: 子任务隔离分析
