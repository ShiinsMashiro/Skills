# Project Ideas (项目创意)

这个技能提供100+编程项目创意，适合不同难度级别的练习。

## When This Skill Applies

This skill activates when the user wants to:
- Find programming project ideas
- Practice coding skills
- Get exercise project suggestions
- Find beginner/intermediate/advanced projects
- Build a portfolio

## Project Categories

### Beginner (初级)
- 计算器 (Calculator)
- 记事本 (Notes App)
- 待办事项 (To-Do App)
- 密码生成器 (Password Generator)
- 天气应用 (Weather App)
- 测验应用 (Quiz App)
- 倒计时器 (Countdown Timer)

### Intermediate (中级)
- 密码管理器 (Password Manager)
- 笔记/闪卡应用 (FlashCards)
- 货币转换器 (Currency Converter)
- 绘图应用 (Drawing App)
- 博客/文章系统 (Blog)
- API天气应用

### Advanced (高级)
- 聊天应用 (Chat App)
- 电商应用 (E-commerce)
- 社交媒体应用 (Social Media)
- 机器学习项目
