# P1: Python编程基础入门

## 完整文本

Hi,We are faculty members at Duke University and we are excited to teach you programming fundamentals for data science. Few fields have generated as much excitement in recent years as data science and its potential to shape the future is immense. In this course and later courses we will provide you with a robust understanding of not just how to program well in Python but also how to work with real data including everything from economic surveys to MRIs. By the time you finish these courses you will not only be able to write test and debug Python programs but also load and manipulate data in a range of formats and summarize plot and model your data. Data science is a fundamentally interdisciplinary field and to be effective a good data scientist must become for manipulating data in all its forms as well as understand the math underlying many statistical and machine learning models and be able to apply these skills to different problems. To deal with this diversity of topics many data science courses enter provide students with a 3 introduction to all the different facets of data science leaving students with a reasonable sense of what the discipline looks like but without providing them with a strong skills set in either programming for data science or statistics. This specialization is different. Instead of providing a superficial introduction to a range of data science topics we have instead chosen to focus on providing you with a robust understanding of programming for data science. This is why Nikonai data scientists have teamed up with experts in teaching programming and software engineering. We will cover the fundamentals of programming and standard Python and also dive into more advanced topics like when and why you should reach for different Python data structures why libraries like NumPy form the backbone of the Python data science ecosystem and how to work with real world data with all its messy complexities. That means that while taking these courses won't make you a data scientist on their own, we've promised that by the end of this specialization even students with zero prior programming experience will feel empowered to take on real world data manipulation challenges. This also makes this specialization an ideal sequence for learners who already have a background in statistics or machine learning but who either unfamiliar with Python because they learn these methods in a language like R or who want to dramatically improve the level of their programming skills. The specialization is also well suited for learners with a background in regular Python interested in becoming familiar with the Python data science ecosystem. Learners in this position may wish to skip some portions of these courses that focus on the standard Python library but will likely find sections on NumPy, pandas, and visualization well suited to their needs. In this course, you will learn how to break down a programming problem using our seven step approach. You will learn an approach to develop an algorithm and test it thoroughly that works with any programming language. Then you will translate your algorithm to Python code, gaining deep knowledge of the language semantics and syntax. Everyone makes mistakes so you will also learn to test programs thoroughly and debug systematically. In later courses, you will learn more advanced data structures in Python and how to use Python libraries like NumPy and pandas to work with large and diverse data sets. Real world data is messy and we will teach you the tools to manage their complexity. Let's dive in and learn some data science.
