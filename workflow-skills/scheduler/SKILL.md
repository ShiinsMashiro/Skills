# Scheduler (定时任务 + 会议系统)

**定时触发 Claude + MiniMax 开会 → 动态调整方向**

## 核心功能

### 1. 定时任务

```
定时触发:
├── 每天 9:00 早会
├── 每周一 计划会
├── 任务完成后 复盘会
└── 自定义时间
```

### 2. Claude + MiniMax 会议

```
会议流程:
├── 🎯 确定议题
├── 🤖 Claude Code 分析现状
├── 🌟 MiniMax 提供建议
├── 📝 制定下一步计划
└── 🔄 调整 auto-pilot 方向
```

### 3. 动态调整方向

```
开会结果:
├── 更新任务优先级
├── 调整执行策略
├── 添加新任务
└── auto-pilot 按新方向执行
```

---

## 会议类型

| 类型 | 触发 | 内容 |
|------|------|------|
| 早会 | 每天 9:00 | 今日任务 |
| 周会 | 每周一 | 本周计划 |
| 复盘会 | 任务完成后 | 总结经验 |
| 方向会 | 项目变更时 | 调整策略 |

---

## 工作流

```
定时触发 (或手动)
     │
     ▼
🎯 确定议题
     │
     ▼
🤖 Claude Code 分析
     │  - 当前进度
     │  - 遇到问题
     │  - 下一步
     │
     ▼
🌟 MiniMax 建议
     │  - 优化方向
     │  - 风险提醒
     │  - 资源分配
     │
     ▼
📝 生成会议纪要
     │  - 决策
     │  - 任务清单
     │  - 优先级
     │
     ▼
🔄 调整 auto-pilot
     │  - 新方向
     │  - 新任务
     │  - 优先级
     │
     ▼
✅ 按新计划执行
```

---

## 命令

| 命令 | 功能 |
|------|------|
| `开会` | 手动触发会议 |
| `早会` | 每日站会 |
| `周会` | 周计划会议 |
| `复盘` | 任务复盘 |
| `定时` | 设置定时任务 |

---

## 自动触发

当检测到：
- 定时时间到达
- 用户说 "开会"
- 任务完成
- 项目方向需要调整

---

*定时开会，动态调整方向！*
