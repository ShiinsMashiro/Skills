# Game Dev Loop (游戏开发循环)

**🎯 NoPUA默认开启**：所有交互都采用爱与尊重的方式！

这是一个自动化游戏开发循环技能。当游戏开发到"终极状态"后，自动深度研究同类游戏，生成新版本迭代。

## When This Skill Applies

This skill activates when:
- 用户说 "循环开发" / "auto develop" / "dev loop"
- 当前项目达到终极状态（所有功能完善）
- 需要自动化持续迭代游戏版本

## 核心功能

### 1. 状态检测 (State Detection)
自动检测游戏开发状态：
- **开发中**: 功能正在实现
- **可测试**: 核心功能完成，可以游玩
- **终极状态**: 所有计划功能完成，需要迭代新版本

### 2. 深度研究 (Deep Research)
当达到终极状态时，自动研究同类游戏：
- 同类型策略游戏研究
- 热门4X游戏分析
- 创新机制调研
- 玩家反馈收集

### 3. 版本迭代 (Version Iteration)
基于研究成果生成新版本：
- V2: 核心玩法增强
- V3: 内容扩展
- V4: 深度机制
- V5+: 持续迭代

### 4. 循环控制
- 检测用户中断信号
- 暂停/恢复功能
- 状态持久化

## 版本规划示例

| 版本 | 名称 | 核心功能 |
|------|------|----------|
| V1 | 基础战斗 | 地图、单位、移动、攻击 |
| V2 | 经济系统 | 金钱、生产、贸易 |

## 深度研究触发条件

当达到以下条件时触发深度研究：
1. 所有P0/P1任务完成
2. 无重大bug
3. 核心玩法完善

## 研究数据源

- Steam 策略游戏排行榜
- 玩家评价分析
- 游戏设计论文
- 类似项目代码分析

## 输出

每次循环生成：
1. 研究报告
2. 新版本功能列表
3. 实现优先级
4. 技术决策
