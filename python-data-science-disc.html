第一章：Python 基础

变量与数据类型

Python 中的变量不需要声明类型，直接赋值即可：

# 整数
age = 25

# 浮点数
price = 19.99

# 字符串
name = "Python"

# 布尔值
is_active = True

# 列表
numbers = [1, 2, 3, 4, 5]

# 字典
person = {"name": "Alice", "age": 30}

运算符

Python 支持多种运算符：

# 算术运算符
+ - * / % ** //

# 比较运算符
== != < > <= >=

# 逻辑运算符
and or not

# 示例
result = (5 + 3) * 2  # 16
is_greater = 10 > 5   # True

第二章：控制流

条件语句

使用 if、elif、else 进行条件判断：

score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "D"

print(f"成绩等级: {grade}")

循环

Python 支持 for 和 while 循环：

# for 循环 - 遍历列表
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# for 循环 - 遍历字典
for key, value in person.items():
    print(f"{key}: {value}")

# while 循环
count = 0
while count < 5:
    print(count)
    count += 1

第三章：函数

定义函数

使用 def 关键字定义函数：

# 基本函数
def greet(name):
    return f"Hello, {name}!"

# 带默认参数的函数
def power(base, exponent=2):
    return base ** exponent

# 可变参数
def sum_all(*args):
    return sum(args)

# 调用
print(greet("Python"))           # Hello, Python!
print(power(3))                  # 9
print(power(2, 3))              # 8
print(sum_all(1, 2, 3, 4, 5))  # 15

练习 1：温度转换 Easy

编写一个函数，将摄氏度转换为华氏度。公式：
F = C × 9/5 + 32

celsius_to_fahrenheit.py

def celsius_to_fahrenheit(celsius):
    return celsius * 9/5 + 32

find_min_max.py

def find_min_max(numbers):
    return (min(numbers), max(numbers))
练习 3：字典统计 Medium

编写一个函数，统计字符串中每个字符出现的次数。

count_chars.py
点击"运行"查看输出...
def count_characters(text):
    result = {}
    for char in text:
        result[char] = result.get(char, 0) + 1
    return result
练习 4：过滤偶数 Medium

编写一个函数，过滤列表中的偶数，只保留奇数。

filter_odd.py
点击"运行"查看输出...
def filter_odds(numbers):
    return [n for n in numbers if n % 2 != 0]
