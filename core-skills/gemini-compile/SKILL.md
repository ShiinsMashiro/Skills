# Gemini 整合器

在并行搜索后，整合并格式化结果。

## 使用场景

| 场景 | 调用 |
|------|------|
| Claude + Gemini 并行搜索后 | 整合结果 |
| 多源结果返回前 | 格式化 |
| 复杂信息 | 分析总结 |

## 工作流程

```
┌─────────────────┐     ┌─────────────────┐
│ Claude Code     │     │ Gemini         │
│ 搜索 PubMed    │     │ 搜索 ArXiv    │
└───────┬─────────┘     └───────┬─────────┘
        │                       │
        └───────────┬───────────┘
                    ▼
        ┌─────────────────────┐
        │  🌟 Gemini 整合     │
        │  - 汇总多源结果    │
        │  - 去重            │
        │  - 格式化          │
        │  - 摘要要点        │
        └─────────┬───────────┘
                  ▼
        ✅ 返回最终结果
```

## 整合功能

- **汇总**: 合并多个搜索源的结果
- **去重**: 识别并合并重复内容
- **格式化**: 统一输出格式 (表格/列表)
- **摘要**: 生成关键要点总结
