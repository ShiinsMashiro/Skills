## 安装方法

### 方法 1: 复制到项目 .claude/skills 目录

```bash
# 克隆仓库
git clone https://github.com/ShiinsMashiro/Skills.git

# 复制到你的项目
cp -r Skills/* your-project/.claude/skills/
```

### 方法 2: 手动安装单个技能

每个技能的安装只需将对应的 `SKILL.md` 文件复制到项目的 `.claude/skills/[skill-name]/` 目录即可。

## 使用方法

### 启动自动驾驶
```
你说: "自动"
Bot: "🎯 请告诉我你想做什么？"
你说: "我要做一个股票分析网站"
Bot: 自动启动工作流...
```

### 直接使用特定技能
```
"帮我生成一个学习'递归'的讨论课页面" → disc-generator
"搜索 Claude Code 相关视频" → god-mode
"规划一个新项目" → project-planner
```

### 技能会自动介入
当你说出任何请求时，**skill-manager** 会自动分析并调用合适的技能。

## Auto-Pilot 权限级别

| 级别 | 名称 | 说明 |
|------|------|------|
| 0 | 手动模式 | 所有权限需确认 |
