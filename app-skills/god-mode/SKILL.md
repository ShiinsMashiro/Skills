# God Mode (超级研究)

**Claude Code + YouTube + NotebookLM + Obsidian = 研究怪兽**

## 核心原理

```
用户: "研究 X"

1. 搜索 YouTube → 视频列表
       ↓
2. NotebookLM 分析 → 深度洞察
       ↓
3. Obsidian 保存 → 知识积累
       ↓
4. 变成你的第二大脑
```

---

## 工作流

### 1. YouTube 搜索
```bash
youtube_search "关键词" --limit 10
```

### 2. NotebookLM 分析
```bash
notebooklm_analyze --source youtube
```

### 3. Obsidian 保存
```bash
obsidian_save --vault "Research" --folder "AI"
```
